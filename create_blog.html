Create a blog

New to blogging?

If you are new to blogging or not familiar with online authoring tools we recommend that you work through the detailed tutorial on how to create a blog before proceeding with the check list below.

Blog tutorial
FAQs

Which blog service?

You can use any blog which has the capabilities to tag or label posts. Most OERu learners use Wordpress or Blogger. Remember to register your blog url on the course site so we can harvest posts using the course tag.

Choosing a blog

Experienced blogger?

If you are familiar with blogs and online authoring tools, proceed with the check list below. If you maintain a regular blog, you are free to use your existing blog and label posts with the course tag: "OCL4Ed"

Register blog

Check list

This check list is intended for learners who are familiar with using online technologies. If you need more detailed instructions, you can delve into a little more detail by following the additional resource links or working sequentially through the subpages of the blog tutorial.

Actions Additional resources

  • Create account on Wordpress.com
  • Create account on Blogger.com

Post your first blog entry and "declare" yourself OERu learners are invited to compete the learning outcome actions referenced in the E-Activities below:
  • E-Activity for Wordpress users; or

Register your blog url Remember to register your blog url on the course site. Tips:
  • Be sure to submit the url for the published view of your blog post and not the url used for editing.
  • Remember to tag or label your post using the course tag: "OCL4Ed".
