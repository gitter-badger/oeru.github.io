Objectives for the orientation session

The purpose of this learning pathway is to set up the technologies you will use to support your learning and to test that everything is working.

Key Points

During this learning pathway you will:

  1. be introduced to the concept of a personal learning environment (PLE);

  2. create your own PLE for this course including a blog and microblog account;

  3. familiarise yourself with the learning environment and where to find the learning resources;

  4. declare yourself to fellow participants using your course blog;

  5. practice microblog posts.
